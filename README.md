# SiaSend - Professional WhatsApp Automation Tool 🚀

SiaSend is a powerful, offline-first desktop application built with Python and Tkinter for automating WhatsApp messaging efficiently and securely.

## ✨ Features
* **Smart Delay System:** Built-in anti-ban mechanisms with randomized delays.
* **Human-like Typing:** Simulates character-by-character typing.
* **Smart Sleep Mode:** Automatically pauses the mission after a set number of messages.
* **Multi-Account Rotation:** Seamlessly switches between different WhatsApp profiles.
* **Database Management:** Built-in duplicate cleaner for Excel/CSV lists.
* **Live Dashboard:** Real-time tracking, UI pie charts, and mission reporting.

## 🛠️ Prerequisites
* Python 3.8+
* Google Chrome browser
